# avantrip-fizzbuzz

Implementar una versión del juego Fizz-Buzz: A partir de array cuyos elementos son del tipo "Integer", generar un nuevo array del mismo tamaño que el original, pero con elementos del tipo “String” que cumplan las siguiente reglas: <br /><br />
● Si el número evaluado es divisible por 3, el resultado debe ser “Fizz” <br />
● Si el número evaluado es divisible por 5, el resultado debe ser “Buzz” <br />
● Si el número evaluado es divisible por 3 y 5, el resultado debe ser “FizzBuzz” <br /> 
● Si el número evaluado no es divisible por 3 ni por 5, el resultado debe ser la representación en String del número  <br /><br />

La resolución debe incluir: <br /><br />

● Documentación mínima que considere necesaria para comunicar el diseño (comentarios, diagramas, etc). <br />
● Código fuente para Java 1.7 <br />
● Implementación de test unitarios si los considera necesarios <br /><br />
